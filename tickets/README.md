# tickets/ : de brokken werk

Na de spec wordt het werk opgeknipt in **tickets** (werkwijze fase 3, template T5). Elk ticket is een dun plakje dat je in een schone sessie kunt bouwen en verifieren.

## Regels
- Een ticket past in een werksessie.
- Het eerste ticket is een dun, end-to-end werkend plakje. Kandidaat: de Make-sandbox connectie-fix (persoonsgebonden connectie vervangen door service-account of bot-token) voor een team-route, plus een foutmelding.
- Elk ticket heeft een "klaar wanneer" met een toetsbare check (test slaagt, screenshot klopt, of een getal haalt een grens).
- Bouw in sandbox `6226897`. Raak het live scenario `3059444` nooit aan zonder expliciete go.

*Nog leeg tot de spec staat en het werk is opgeknipt.*
